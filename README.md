# Reproducibility package for public deposit

This package is prepared for upload to a public repository such as Zenodo,
Figshare, or GitHub with Zenodo archiving before manuscript submission.

## Contents

- `experiments/`: preprocessing, model training, evaluation, aggregation, and
  figure-generation scripts.
- `experiments/adapters/`: dataset conversion scripts for UCSD, StoreNet, and
  Bath public datasets.
- `experiments/src/pvexp/`: model, metric, data, physics, and reporting modules.
- `experiments/results/`: compact CSV summaries and LaTeX tables used to
  regenerate the manuscript results.
- `data/processed/*.metadata.json`: metadata generated by each dataset adapter.

The raw public datasets are not redistributed here. They should be downloaded
from the original public repositories listed in the manuscript Data availability
statement.

## Public raw data sources

- UCSD Microgrid Database: https://github.com/sushilsilwal3/UCSD-Microgrid-Database
- UCSD data descriptor: https://doi.org/10.1063/5.0038650
- StoreNet dataset: https://doi.org/10.6084/m9.figshare.c.6829134
- University of Bath PV-battery microgrid dataset: https://doi.org/10.15125/BATH-01287

## Before submission

1. Upload this package to a permanent public repository.
2. Obtain a DOI from the repository.
3. Replace `https://doi.org/TO-BE-ASSIGNED` in the manuscript Code availability
   statement with the assigned DOI.
4. If a GitHub repository is used first, create a release and archive that
   release through Zenodo so that the DOI points to an immutable version.
